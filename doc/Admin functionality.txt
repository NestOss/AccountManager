Варианты использования сервиса администратором:
1.Просмотр и редактирование списка организаций. При переименовании организации должны вноситься
изменения в данные пользователей - членов организации.

2.Просмотр и редактирование списка ВСЕХ пользователей. В списке должны выводиться:
 - идентификатор (логин) пользователя;
 - статус пользователя (активен/блокирован)
 - полное имя пользователя;
 - роль пользователя;
 - организация, к которой принадлежит пользователь.

    При создании нового пользователя вводятся все данные, включая пароль. Полное имя пользователя
вводится в двух полях - имя и фамилия. Имя организации выбирается только из списка.
    При редактировании пользователя можно изменить его полное имя, роль, организацию и статус.
Имя организации выбирается только из списка.
    Должна быть предусмотрена функция задания нового пароля пользователя.
    На странице должен быть предусмотрен фильтр пользователей по идентификатору, полному имени, роли,
организации, статусу. Должен быть организован постраничный просмотр пользователей. При применении фильтров должен
генерироваться новый набор страниц.

Все действия по изменению информации должны записываться в лог-файл (пароли в лог-файл не записываются).
