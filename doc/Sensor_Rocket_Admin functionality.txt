Варианты использования сервиса администратором в организации:
1.Просмотр и редактирование списка пользователей (исключая администраторов) своей организации.
 В списке должны выводиться:
 - идентификатор (логин) пользователя;
 - статус пользователя (активен/блокирован)
 - полное имя пользователя.

    При создании нового пользователя вводятся логин/пароль и полное имя пользователя 
(в двух полях - имя и фамилия). Имя организации пользователя присваивается равным имени организации
администратора. Роль пользователя определяется на основании роли администратора.
    При редактировании пользователя можно изменить его полное имя и статус. Должна быть
предусмотрена функция задания нового пароля пользователя.
    На странице должен быть предусмотрен фильтр пользователей по идентификатору, полному имени, статусу.

Все действия по изменению информации должны записываться в лог-файл (пароли в лог-файл не записываются).
